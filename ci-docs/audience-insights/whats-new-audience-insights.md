# <a name="whats-new-in-the-audience-insights-capability-of-dynamics-365-customer-insights"></a>มีอะไรใหม่ในความสามารถข้อมูลเชิงลึกกลุ่มเป้าหมายของ Dynamics 365 Customer Insights

[!INCLUDE [cc-data-platform-banner](../includes/cc-data-platform-banner.md)]

พวกเรารู้สึกตื่นเต้นที่จะประกาศการปรับปรุงใหม่ล่าสุดของเรา! บทความนี้สรุปคุณลักษณะตัวอย่างสำหรับสาธารณะ การปรับปรุงความพร้อมใช้งานทั่วไป และการอัปเดตคุณลักษณะ เมื่อต้องการดูแผนคุณลักษะในระยะยาว ให้ดูที่ [แผนการนำออกใช้ของ Dynamics 365 และ Power Platform](/dynamics365/release-plans/)

เราเปิดตัวการอัปเดตตามภูมิภาค ดังนั้น บางภูมิภาคอาจเห็นคุณลักษณะก่อนภูมิภาคอื่นๆ หากไม่ได้ระบุไว้แตกต่างกัน คุณไม่จำเป็นต้องดำเนินการใดๆ และเราจะอัปเดตแอปโดยอัตโนมัติ โดยไม่ต้องหยุดทำงาน

> [!TIP]
> เมื่อต้องการ ส่งและลงคะแนนให้กับ การร้องขอคุณลักษณะ และ คำแนะนำผลิตภัณฑ์ ไปที่ [พอร์ทัลแนวคิดเกี่ยวกับโปรแกรมประยุกต์ Dynamics 365](https://experience.dynamics.com/ideas/categories/?forum=79a8c474-4e35-e911-a971-000d3a4f3343&forumName=Dynamics%20365%20Customer%20Insights)

## <a name="april-2021-updates"></a>อัปเดต เมษายน 2021

การอัปเดตในเดือนเมษายน 2021 ประกอบด้วยคุณลักษณะ การอัปเกรดประสิทธิภาพ และการแก้ไขข้อบกพร่องต่างๆ

### <a name="data-unification"></a>การรวมข้อมูล
 
- **ประสบการณ์การผสานขั้นสูงสำหรับการรวมข้อมูล**    
  
   ขณะนี้เรามีประสบการณ์ของผู้ใช้ขั้นสูงในการตั้งค่าคอนฟิกการผสานของกระบวนการรวมข้อมูล การเปลี่ยนแปลงรวมถึงการจัดลำดับที่ใช้งานง่ายของฟิลด์ที่ผสานและสถิติโดยละเอียดเกี่ยวกับฟิลด์แบบรวมและแบบเดี่ยว

- **การจัดลำดับเอนทิตีใหม่และตั้งค่าคอนฟิกเรกคอร์ดแหล่งที่มาทั้งหมดลงในเอนทิตีลูกค้า**  
      
   ตอนนี้คุณสามารถจัดลำดับใหม่และลบเอนทิตีออกจากแผนการรวมกลุ่มที่มีอยู่ในกระบวนการรวมข้อมูลได้ ซึ่งให้ความยืดหยุ่นในการจัดลำดับเอนทิตีใหม่ในกระบวนการจับคู่ตามความต้องการทางธุรกิจ นอกจากนี้ เรายังเปิดใช้งานรวมเรกคอร์ดที่ไม่ตรงกันทั้งหมดไว้ในเอนทิตี *ลูกค้า* สุดท้าย ซึ่งช่วยให้พวกเขาสามารถกำหนดนิยามชุดข้อมูลโปรไฟล์ลูกค้าของพวกเขาได้

### <a name="enrichments"></a>การเพิ่มความสมบูรณ์

 - **การเพิ่มความสมบูรณ์ใหม่: ที่อยู่ที่ปรับปรุงแล้ว**    
  
   เรามีความยินดีที่จะแนะนำการเพิ่มความสมบูรณ์ใหม่เพื่อเพิ่มความสมบูรณ์ที่อยู่ในข้อมูลลูกค้าของคุณ ที่อยู่ในข้อมูลของคุณอาจไม่มีโครงสร้าง ไม่สมบูรณ์ หรือไม่ถูกต้อง คุณลักษณะนี้ใช้โมเดลของ Microsoft เพื่อกำหนดมาตรฐานและเพิ่มความสมบูรณ์ที่อยู่ของคุณลงในรูปแบบ Common Data Model เพื่อความแม่นยำและข้อมูลเชิงลึกที่ดีขึ้น
 
   สำหรับข้อมูลเพิ่มเติม โปรดดู [การเพิ่มความสมบูรณ์ของโปรไฟล์ลูกค้าด้วยที่อยู่ที่ปรับปรุงแล้ว](enrichment-enhanced-addresses.md)

- **ประสบการณ์การตั้งค่าคอนฟิกที่แนะนำสำหรับการเพิ่มความสมบูรณ์**    
  
   เราได้ทบทวนประสบการณ์การตั้งค่าคอนฟิกสำหรับการเพิ่มความสมบูรณ์ด้วยประสบการณ์ที่เรียบง่ายและที่แนะนำ ตอนนี้คุณมีกระบวนการที่ชัดเจนทีละขั้นตอนสำหรับการสร้างและการแก้ไขการเพิ่มความสมบูรณ์
 
   นอกจากนี้ เรายังแยกการตั้งค่าคอนฟิกของการเชื่อมต่อสำหรับการเพิ่มความสมบูรณ์ของบุคคลที่สาม เพื่อให้สามารถใช้การเชื่อมต่อเดียวกันโดยการเพิ่มความสมบูรณ์หลายรายการ เฉพาะผู้ดูแลระบบเท่านั้นที่สามารถตั้งค่าคอนฟิกการเชื่อมต่อใหม่ แต่การเชื่อมต่อที่สร้างขึ้นจะพร้อมใช้งานสำหรับทั้งผู้ดูแลระบบและผู้สนับสนุน    

   สำหรับข้อมูลเพิ่มเติม ดู [ภาพรวมของการเชื่อมต่อ](connections.md)

- **การเพิ่มข้อมูลหลายรายการในชนิดเดียวกัน**    
  
   ตอนนี้เราอนุญาตให้ผู้ใช้สร้างและจัดการการเพิ่มความสมบูรณ์หลายรายการในชนิดการเพิ่มความสมบูรณ์เดียวกัน ตัวอย่างเช่น ตอนนี้คุณสามารถสร้างการเพิ่มความสมบูรณ์ที่อยู่สองรายการแยกกัน เพื่อการเพิ่มความสมบูรณ์เซ็กเมนต์ลูกค้าที่แตกต่างกันสองเซ็กเมนต์ ข้อจำกัดมีผลกับจำนวนการเพิ่มความสมบูรณ์ชนิดเดียวกันที่สามารถสร้างได้ และแตกต่างกันไปโดยขึ้นอยู่กับชนิดการเพิ่มความสมบูรณ์
  
   สำหรับข้อมูลเพิ่มเติม โปรดดู [การเพิ่มข้อมูลสำหรับโปรไฟล์ลูกค้า](enrichment-hub.md)

## <a name="march-2021-updates"></a>การปรับปรุงเดือนมีนาคม 2021

การปรับปรุงในเดือนมีนาคม 2021 ประกอบด้วยคุณสมบัติต่าง ๆ การอัปเกรดประสิทธิภาพ และการแก้ไขข้อบกพร่อง

### <a name="activities"></a>กิจกรรม

- **ตัวช่วยสร้างกิจกรรมและชนิดความหมาย**

   เราได้ปรับปรุงและอัปเดตประสบการณ์การแม็ปกิจกรรมของเรา เพื่อเป็นแนวทางและลดความซับซ้อนในการสร้างการแม็ปกิจกรรม ในประสบการณ์ใหม่นี้ ผู้ใช้จะได้รับประสบการณ์ที่แนะนำเพื่อช่วยทำแต่ละขั้นตอนของกระบวนการให้เสร็จสิ้น ในขั้นตอนการแมปกิจกรรม นอกเหนือจากการเลือกชนิดกิจกรรมต่าง ๆ แล้ว ผู้ใช้ยังสามารถเลือกที่จะแมปข้อมูลสำหรับความหมายได้อีกด้วยสำหรับ *การสมัครสมาชิก* และ/หรือ *SalesOrderLine* เป็นสคีมามาตรฐานอุตสาหกรรม ซึ่งสามารถใช้สำหรับการบริโภคดาวน์สตรีม   

   สำหรับข้อมูลเพิ่มเติม ดู [กิจกรรมของลูกค้า](activities.md)

### <a name="data-ingestion"></a>การรับข้อมูล

- **เชื่อมต่อกับแหล่งข้อมูลภายในองค์กรโดยใช้โฟลว์ข้อมูลและเกตเวย์ Power Platform** เรายินดีที่จะประกาศตัวอย่างของโฟลว์ข้อมูล Power Platform และการเชื่อมต่อภายในองค์กรโดยใช้เกตเวย์ใน Customer Insights ที่เชื่อมโยงสภาพแวะล้อม Power Platform หรือ Dataverse แหล่งข้อมูลใหม่ใด ๆ ที่สร้างขึ้นในสภาพแวดล้อม Customer Insights ที่มีการเชื่อมโยงสภาพแวดล้อม Dataverse จะเริ่มต้นเป็นโฟลว์ข้อมูล Power Platform ที่นำการเชื่อมต่อข้อมูลภานในองค์กรและชุดตัวเชื่อมต่อและความสามารถในการแปลง

### <a name="extensibility"></a>ความสามารถในการเพิ่ม

- **การส่งออกที่จัดระเบียบในการเชื่อมต่อและการส่งออก** เราได้เปลี่ยนชื่อของหน้า **ปลายทางการส่งออก** เป็น **การเชื่อมต่อ** และเพิ่มหน้าแยกสำหรับ **การส่งออก** ในการอัปเดตนี้ เราจะเปลี่ยนการส่งออกที่มีอยู่เป็นคู่ของการเชื่อมต่อและการส่งออกโดยใช้การเชื่อมต่อนั้น ขณะนี้ผู้ดูแลระบบมีความชัดเจนมากขึ้นเกี่ยวกับข้อมูลขาออกในหน้า **การเชื่อมต่อ** บทบาทของผู้ใช้ทั้งหมดสามารถเข้าถึงหน้า **การส่งออก** แต่เฉพาะผู้ดูแลระบบเท่านั้นที่สามารถเลือกที่จะอนุญาตให้ผู้ร่วมให้ข้อมูลแก้ไขการส่งออกเฉพาะด้วยการเชื่อมต่อที่แบ่งปันได้     
  สำหรับข้อมูลเพิ่มเติม โปรดดู [ภาพรวมการเชื่อมต่อ](connections.md) และ [ภาพรวมการส่งออก](export-destinations.md)

- **ส่งออกเซ็กเมนต์ไปยัง Campaign Monitor** เราได้ขยายปลายทางการส่งออกของเราเพื่อรวม Campaign Monitor ตอนนี้คุณสามารถส่งออกเซ็กเมนต์จาก Customer Insights ไปยังรายการ Campaign Monitor และใช้เป็นพื้นฐานสำหรับแคมเปญการตลาดของคุณได้    
   สำหรับข้อมูลเพิ่มเติม โปรดดู [ส่งออกไปยัง Campaign Monitor](export-campaign-monitor.md)

- **ส่งออกเซ็กเมนต์ไปยัง Constant Contact** เราได้ขยายปลายทางการส่งออกของเราเพื่อรวม Constant Contact ตอนนี้คุณสามารถส่งออกเซ็กเมนต์จาก Customer Insights ไปยังรายการ Constant Contact และใช้เป็นพื้นฐานสำหรับแคมเปญการตลาดของคุณได้   
   สำหรับข้อมูลเพิ่มเติม โปรดดู [ส่งออกไปยัง Constant Contact](export-constant-contact.md)

- **ส่งออกเซ็กเมนต์ไปยัง RollWorks** เราได้ขยายปลายทางการส่งออกของเราเพื่อรวม RollWorks ตอนนี้คุณสามารถส่งออกเซ็กเมนต์จาก Customer Insights ไปยังผู้ชมบน RollWorks และใช้เป็นพื้นฐานสำหรับโฆษณา B2B ของคุณได้    
   สำหรับข้อมูลเพิ่มเติม โปรดดู [ส่งออกข้อมูลไปยัง RollWorks ](export-rollworks.md)

- **ส่งออกเซ็กเมนต์ไปยัง Snapchat** เราได้ขยายปลายทางการส่งออกของเราเพื่อรวม Snapchat ตอนนี้คุณสามารถส่งออกเซ็กเมนต์จาก Customer Insights ไปยังผู้ชมบน Snapchat และใช้เป็นพื้นฐานสำหรับโฆษณาของคุณได้     
   สำหรับข้อมูลเพิ่มเติม โปรดดู [ส่งออกข้อมูลไปยัง Snapchat](export-snapchat.md)

### <a name="predictions"></a>การคาดคะเน

- **ใช้ตัวกรองผลิตภัณฑ์ในคำแนะนำผลิตภัณฑ์เชิงคาดคะเน** เราได้เพิ่มความสามารถในการใช้ตัวกรองผลิตภัณฑ์ในรูปแบบการแนะนำผลิตภัณฑ์ของเรา ตอนนี้คุณสามารถสร้างการคาดคะเนที่ใช้เฉพาะส่วนย่อยของผลิตภัณฑ์ของคุณ    
   สำหรับข้อมูลเพิ่มเติม โปรดดู [กำหนดค่าตัวกรองผลิตภัณฑ์](predict-product-recommendation.md#configure-product-filters)

- **สร้างเซ็กเมนต์จากการคาดคะเนแบบจำลอง** เราได้เพิ่มวิธีที่รวดเร็วในการสร้างเซ็กเมนต์โดยใช้ผลลัพธ์ของแบบจำลองการคาดคะเน จากหน้าผลลัพธ์ของแบบจำลอง คุณสามารถสร้างเซ็กเมนต์ใหม่ได้อย่างง่ายดายโดยเลือกตัวเลือก **สร้างเซ็กเมนต์** ใหม่    
  สำหรับข้อมูลเพิ่มเติม โปรดดู [สร้างเซ็กเมนต์ตามแบบจำลองการคาดคะเน](prediction-based-segment.md)

- **คำอธิบายสำหรับคำแนะนำผลิตภัณฑ์** เราได้เพิ่มข้อมูลที่อธิบายถึงปัจจัยสำคัญที่ได้เรียนรู้จากโมเดล AI เพื่อสร้างคำแนะนำผลิตภัณฑ์และระดับที่ปัจจัยเหล่านั้นมีส่วนต่อคำแนะนำผลิตภัณฑ์ ข้อมูลนี้จะถูกเพิ่มลงในหน้าจอผลลัพธ์ของแบบจำลอง    
   สำหรับข้อมูลเพิ่มเติม โปรดดู [ตรวจสอบสถานะการคาดคะเนและผลลัพธ์](predict-product-recommendation.md#review-a-prediction-status-and-results)

## <a name="february-2021-updates"></a>การปรับปรุงเดือนกุมภาพันธ์ 2021

การปรับปรุงในเดือนกุมภาพันธ์ 2021 ประกอบด้วยคุณลักษณะต่างๆ การอัปเกรดประสิทธิภาพ และการแก้ไขข้อบกพร่อง

#### <a name="extensibility"></a>ความสามารถในการเพิ่ม

- **ส่งออกเซ็กเมนต์ไปยัง AdRoll**

  เราได้ขยายปลายทางการส่งออกของเราเพื่อรวม AdRoll ตอนนี้คุณสามารถส่งออกเซ็กเมนต์จาก Customer Insights ไปยังกลุ่มเป้าหมาย AdRoll และใช้เป็นข้อมูลพื้นฐานสำหรับการโฆษณาของคุณได้ สำหรับข้อมูลเพิ่มเติม โปรดดู [ตัวเชื่อมต่อสำหรับ AdRoll](export-adroll.md)

#### <a name="segments"></a>เซ็กเมนต์
 
- **ทำซ้ำเซ็กเมนต์**
  
  หากต้องการสร้างเซ็กเมนต์ใหม่ตามเซ็กเมนต์ที่มีอยู่ ตอนนี้คุณสามารถทำซ้ำเซ็กเมนต์และแก้ไขเซ็กเมนต์ที่ซ้ำกันเพื่อปรับปรุงเพิ่มเติมได้ 

- **เพิ่มแอตทริบิวต์เพิ่มเติมให้กับเซ็กเมนต์**

  ตอนนี้ คุณสามารถรวมแอตทริบิวต์ในผลลัพธ์ของเซ็กเมนต์ของคุณได้ แม้ว่าแอตทริบิวต์เหล่านี้จะไม่ได้เป็นส่วนหนึ่งของโปรไฟล์ลูกค้าก็ตาม ตัวอย่างเช่น รวมรหัสการสมัครสมาชิกในเซ็กเมนต์แม้ว่าจะเป็นส่วนหนึ่งของเอนทิตีการสมัครสมาชิกที่มีความสัมพันธ์แบบ M:1 กับเอนทิตีลูกค้า ตราบเท่าที่แอตทริบิวต์เป็นของเอนทิตีที่เกี่ยวข้องกับเอนทิตีลูกค้า คุณสามารถรวมแอตทริบิวต์เหล่านี้ได้  

#### <a name="predictions"></a>การคาดคะเน

- **สร้างคำแนะนำผลิตภัณฑ์ที่คาดคะเนได้**

  การทำความเข้าใจว่าลูกค้าสนใจซื้ออะไรเป็นขั้นตอนแรกที่จำเป็นในการปรับปรุงรายได้ของธุรกิจและสร้างความภักดีของลูกค้าผ่านการปรับเปลี่ยนตามแบบเฉพาะและการมีส่วนร่วม การให้คำแนะนำสำหรับผลิตภัณฑ์ที่ไม่สอดคล้องกับความสนใจของลูกค้าสามารถสร้างความรู้สึกตัดสัมพันธ์ระหว่างลูกค้ากับธุรกิจของคุณ และในที่สุดก็จำกัดรายได้และประสบการณ์โดยรวมที่เป็นไปได้สำหรับลูกค้า 

  ด้วยการใช้ข้อมูลของคุณเอง ตอนนี้คุณสามารถสร้างการคาดคะเนสำหรับผลิตภัณฑ์ที่ลูกค้าของคุณมีแนวโน้มที่จะซื้อในอนาคตได้แล้ว สำหรับข้อมูลเพิ่มเติม โปรดดู [การคาดคะเนคำแนะนำผลิตภัณฑ์](predict-product-recommendation.md)

#### <a name="system-administration"></a>ผู้ดูแลระบบ

- **การคัดลอกสภาพแวดล้อมรองรับแหล่งข้อมูลประเภทต่างๆ มากขึ้น**

  ผู้ดูแลระบบสามารถคัดลอกการกำหนดค่าสภาพแวดล้อมไปยังสภาพแวดล้อมใหม่ในองค์กรเดียวกัน คุณลักษณะนี้จะขยายฟังก์ชันการทำงานของการคัดลอกสภาพแวดล้อมสำหรับกรณีที่แหล่งข้อมูลอิงอยู่กับที่จัดเก็บข้อมูลดิบ Common Data Service หรือมีการใช้โฟลเดอร์ Common Data Model

## <a name="january-2021-updates"></a>การอัปเดตประจำเดือนมกราคม 2021

การปรับปรุงในเดือนมกราคม 2021 ประกอบด้วยคุณลักษณะต่างๆ การอัปเกรดประสิทธิภาพ และการแก้ไขข้อบกพร่อง

#### <a name="extensibility"></a>ความสามารถในการเพิ่ม

- **ฟังก์ชันเพิ่มเติมและประสิทธิภาพที่เพิ่มขึ้นสำหรับการส่งออก SFTP** ตอนนี้คุณสามารถส่งออกเอนทิตีผลลัพธ์ทั้งหมดจาก Customer Insights ไปยังโฮสต์ SFTP ได้ ก่อนหน้านี้ การส่งออกถูกจำกัดไว้ที่เอนทิตีเซ็กเมนต์ นอกจากนี้ ประสิทธิภาพของการส่งออก SFTP ยังช่วยให้ปริมาณข้อมูลมากขึ้นโดยใช้เวลาน้อยลงโดยขึ้นอยู่กับประสิทธิภาพของโฮสต์ SFTP ของคุณ    
  สำหรับข้อมูลเพิ่มเติม โปรดดู [ตัวเชื่อมต่อสำหรับ SFTP (ตัวอย่าง)](export-sftp.md)  

#### <a name="segments"></a>เซ็กเมนต์

- **การเรียนรู้เกี่ยวกับเครื่องที่ดำเนินการโดยเซ็กเมนต์ที่แนะนําเพื่อปรับปรุงเมตริก** มีวิธีใหม่ในการค้นหาและสร้างเซ็กเมนต์ ระบบใช้แบบจำลอง AI เพื่อแนะนำเซ็กเมนต์ที่สามารถช่วยปรับปรุง KPI (การวัดผล) ที่คุณกำลังติดตามอยู่ เราแสดงขอบเขตของอิทธิพลของแอตทริบิวต์ที่คุณเลือกในการวัดหรือแอตทริบิวต์หลักอื่น ๆ ข้อมูลนี้ช่วยในการค้นหาเซ็กเมนต์ที่เป็นไปได้ที่นำเสนอโอกาส    
  สำหรับข้อมูลเพิ่มเติม โปรดดู [เซ็กเมนต์แนะนํา (ตัวอย่าง)](suggested-segments.md)

#### <a name="data-unification"></a>การรวมข้อมูล

- **ประสบการณ์การจับคู่ที่เพิ่มขึ้น** ในพื้นที่การรวมข้อมูล ประสบการณ์การจับคู่ได้รับการอัปเดต ช่วยให้คุณกำหนดค่าและดูกฎการจับคู่ รวมถึงสถิติโดยละเอียดเพื่ออธิบายเพิ่มเติมว่าการจับคู่ทำงานอย่างไร มีตัวเลือกในการปิดใช้กฎการจับคู่ ดังนั้นจึงไม่ทำงานอีกต่อไป ในขณะที่ยังคงการกำหนดค่า ลากและวางกฎการจับคู่ และอื่น ๆ
  สำหรับข้อมูลเพิ่มเติม โปรดดูที่ [จับคู่เอนทิตี](match-entities.md)

- **ผลลัพธ์การลบข้อมูลซ้ำซ้อนจากกระบวนการจับคู่พร้อมใช้งานในรูปแบบเอนทิตี** ขณะนี้ผลลัพธ์ของกระบวนการคัดลอกข้อมูลซ้ำซ้อนจากกระบวนการจับคู่ถูกเขียนลงในเอนทิตีแยกต่างหากสำหรับการวิเคราะห์เพิ่มเติม เอนทิตีนี้ประกอบด้วยฟิลด์ที่ใช้ในกระบวนการขจัดข้อมูลซ้ำซ้อนและเรกคอร์ดผู้ชนะและเรกคอร์ดสำรองที่เกี่ยวข้องซึ่งรวมเข้ากับเรกคอร์ดผู้ชนะ
  สำหรับข้อมูลเพิ่มเติม โปรดดู [ผลลัพธ์การขจัดข้อมูลซ้ำซ้อนเป็นเอนทิตี](match-entities.md#deduplication-output-as-an-entity)

#### <a name="system-administration"></a>ผู้ดูแลระบบ

- **แบ่งปันข้อมูลไปยัง Microsoft Dataverse อย่างราบรื่น** ตอนนี้คุณสามารถแชร์ผลลัพธ์ Customer Insights กับโปรแกรมประยุกต์ Microsoft Dataverse โดยใช้ Data Lake ที่จัดการ Microsoft Dataverse เมื่อคุณเชื่อมโยงสภาพแวดล้อม Dataverse ด้วย Customer Insights คุณจะได้รับตัวเลือกในการเปิดใช้งานการแบ่งปันข้อมูล
  สำหรับข้อมูลเพิ่มเติม ดู [จัดการการสภาพแวดล้อม](manage-environments.md)
